## Overview
<br>

Voyager is a robust full-stack application built to bridge a highly performant backend infrastructure with a fluid, responsive frontend. The platform provides a seamless environment for users to create personalized accounts, select and explore global destinations, manage complex travel itineraries, and track expenses in real-time. 

Built with a focus on high availability, secure data architecture, and intuitive user experience, Voyager serves as a centralized hub for modern travel planning.

&nbsp;

&nbsp;

## Key Features
<br>

* **Secure Account Management:** Full registration and authentication system with role-based access control, JWT session management, and encrypted credentials.
* **Interactive Destination Selection:** Dynamic interfaces allowing users to browse, search, and pin travel locations globally with immediate state updates.
* **Comprehensive Expense Tracking:** Built-in financial management tools to allocate budgets, record granular travel expenses, and monitor costs across multiple active trips.
* **Personalized Travel Dashboard:** A customized, analytical user interface displaying upcoming journeys, saved destinations, and graphical travel statistics.
* **Responsive & Adaptive Design:** Strictly optimized viewing experience across desktop, tablet, and mobile environments without compromising functionality.

## Technology Stack
<br>

The application utilizes a modern, decoupled architecture to ensure maximum cohesion and scalability between the client interface and server logic.

| Layer | Primary Technology | Description |
| :--- | :--- | :--- |
| **Frontend** | React, TypeScript, SCSS | Handles the interactive UI, component state management, and client-side routing. |
| **Backend** | C#, .NET Web API | Serves as the core engine processing business rules, validation, and RESTful endpoints. |
| **Database** | PostgreSQL / SQL Server | Relational database management system ensuring robust and secure data persistence. |
| **ORM** | Entity Framework Core | Facilitates code-first object-relational mapping, data querying, and schema migrations. |

## System Architecture
<br>

Voyager operates on a secure multi-tier architecture, ensuring separation of concerns:

1. **Presentation Tier (React):** Manages user inputs, global application state, and asynchronous API communication.
2. **Application Tier (C# / .NET):** Validates incoming payload data, executes travel and financial logic, and enforces strict authorization policies.
3. **Data Tier:** Safely handles high-volume transactions regarding user profiles, geographical coordinates, and financial records.
