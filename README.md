VOYAGER
Interactive Travel & Expense Management PlatformOverviewVoyager is a comprehensive, interactive full-stack application designed to revolutionize how users plan, manage, and experience their travels. By bridging a robust backend infrastructure with a highly responsive frontend, the application provides a seamless environment for users to create personalized accounts, select global destinations, manage travel expenses, and organize their itineraries in an interactive digital workspace.Built with a focus on performance, scalability, and user experience, Voyager serves as a centralized hub for all travel-related planning and tracking.Key FeaturesSecure Account Management: Full registration and authentication system with role-based access control and encrypted credentials.Interactive Destination Selection: Dynamic interfaces allowing users to browse, search, and select travel locations globally with real-time feedback.Comprehensive Expense Tracking: Built-in financial management tools to allocate budgets, record travel expenses, and monitor costs across different trips.Personalized Travel Dashboard: A customized user interface displaying upcoming journeys, saved destinations, and overall travel statistics.Responsive Design: Optimized viewing experience across desktop, tablet, and mobile environments.Technology StackThe application is built using a modern, decoupled architecture to ensure high cohesion and low coupling between the client and server.LayerTechnologies UsedDescriptionFrontendReact, JavaScript/TypeScript, CSS/SCSSHandles the interactive user interface and client-side routing.BackendC#, .NET Web APIServes as the core engine processing business logic and RESTful API endpoints.DatabasePostgreSQL / SQL ServerRelational database management system for secure data persistence.ORMEntity Framework CoreFacilitates object-relational mapping and database migrations.System ArchitectureVoyager operates on a multi-tier architecture:Presentation Tier (React): Manages state, user inputs, and asynchronous API calls.Application Tier (C# / .NET): Validates incoming data, processes travel/expense logic, and enforces security policies.Data Tier: Safely stores user profiles, destination coordinates, and transaction records.Getting StartedFollow these instructions to set up the Voyager application in your local development environment.PrerequisitesEnsure you have the following installed on your local machine:.NET SDK (version 6.0 or higher)Node.js (version 16.x or higher)A relational database (PostgreSQL or SQL Server)GitBackend Setup (C#)Clone the repository and navigate to the backend directory:Bashgit clone https://github.com/your-username/voyager.git
cd voyager/backend
Restore the necessary .NET packages:Bashdotnet restore
Update the appsettings.json file with your specific database connection string.Apply the latest database migrations:Bashdotnet ef database update
Run the backend server:Bashdotnet run
The API will typically be available at http://localhost:5000 or https://localhost:5001.Frontend Setup (React)Open a new terminal instance and navigate to the frontend directory:Bashcd voyager/frontend
